# 롤 내전 디코봇 설계도 (v2)

## 기술 스택

| 항목 | 내용 |
|---|---|
| 언어 | Python 3.11+ |
| Discord 라이브러리 | discord.py 2.x (슬래시 커맨드 / 버튼 / 모달 / Select 지원) |
| 데이터베이스 | SQLite + aiosqlite (비동기) |
| 외부 API | Riot Games API (KR 서버, Asia 리전) |
| 환경변수 관리 | python-dotenv |

---

## 파일 구조

```
롤 내전 디코봇/
├── .env                    ← 실제 토큰·API 키 (git 제외)
├── .env.example            ← 환경변수 템플릿
├── requirements.txt        ← 패키지 목록
├── config.py               ← 설정값 모음
├── database.py             ← SQLite CRUD 함수
├── riot_api.py             ← Riot API 호출 / 데이터 파싱 / 이모지
├── main.py                 ← 봇 진입점 + 백그라운드 태스크
└── cogs/
    ├── lobby.py            ← 내전 커맨드 + 포지션 선택 + 대기열
    ├── admin.py            ← 관리자 전용 커맨드 그룹
    ├── team.py             ← /팀짜기 (자동 밸런싱)
    └── profile.py          ← /아이디변경, /내정보, /봇상태
```

## DB 스키마

### users
| 컬럼 | 타입 | 설명 |
|---|---|---|
| discord_id | TEXT (PK) | 디스코드 유저 ID |
| puuid | TEXT (UNIQUE) | 라이엇 PUUID |
| game_name | TEXT | 게임 이름 |
| tag_line | TEXT | 태그라인 |
| main_role | TEXT | 주라인 |
| sub_role | TEXT | 부라인 |
| solo_tier / rank / lp | TEXT / TEXT / INT | 솔로랭크 |
| flex_tier / rank / lp | TEXT / TEXT / INT | 자유랭크 |
| updated_at | TIMESTAMP | 마지막 갱신 |

### lobbies
| 컬럼 | 타입 | 설명 |
|---|---|---|
| id | INTEGER (PK) | 내전 번호 |
| guild_id | TEXT | 서버 ID |
| creator_discord_id | TEXT | 개설자 ID |
| channel_id | TEXT | 채널 ID |
| message_id | TEXT | 모집 메시지 ID |
| status | TEXT | open / full / cancelled |
| created_at | TIMESTAMP | 개설 시각 |

### lobby_members *(position 컬럼 추가)*
| 컬럼 | 타입 | 설명 |
|---|---|---|
| lobby_id | INTEGER (PK, FK) | 내전 ID |
| discord_id | TEXT (PK) | 참가자 ID |
| position | TEXT | 선택 포지션 |
| joined_at | TIMESTAMP | 참가 시각 |

### waitlist *(신규)*
| 컬럼 | 타입 | 설명 |
|---|---|---|
| id | INTEGER (PK) | 고유 번호 |
| lobby_id | INTEGER (FK) | 내전 ID |
| discord_id | TEXT | 대기자 ID |
| position | TEXT | 희망 포지션 |
| joined_at | TIMESTAMP | 대기 등록 시각 |

### server_settings *(신규)*
| 컬럼 | 타입 | 설명 |
|---|---|---|
| guild_id | TEXT (PK) | 서버 ID |
| alert_role_id | TEXT | 내전 알림 역할 ID |
| log_channel_id | TEXT | 에러·이벤트 로그 채널 ID |
| cooldown_minutes | INTEGER | 내전 취소 후 재개설 쿨타임 (분) |
| max_members | INTEGER | 내전 최대 인원 (기본 10) |

### cooldowns *(신규)*
| 컬럼 | 타입 | 설명 |
|---|---|---|
| discord_id | TEXT (PK) | 유저 ID |
| last_cancel_at | TIMESTAMP | 마지막 취소 시각 |

### banned_users *(신규)*
| 컬럼 | 타입 | 설명 |
|---|---|---|
| guild_id | TEXT (PK) | 서버 ID |
| discord_id | TEXT (PK) | 차단된 유저 ID |
| banned_until | TIMESTAMP | 차단 만료 시각 (NULL = 영구) |
| reason | TEXT | 차단 사유 |

### scheduled_lobbies *(신규)*
| 컬럼 | 타입 | 설명 |
|---|---|---|
| id | INTEGER (PK) | 고유 번호 |
| creator_discord_id | TEXT | 개설자 ID |
| guild_id | TEXT | 서버 ID |
| channel_id | TEXT | 채널 ID |
| fire_at | TIMESTAMP | 개설 예약 시각 |
| status | TEXT | pending / done / cancelled |

## 슬래시 커맨드

### 일반 커맨드

| 커맨드 | 설명 |
|---|---|
| `/내전시작` | 내전 개설 (1인 1개 / 쿨타임 / 알림 역할 태그) |
| `/내전취소` | 내 내전 취소 (참가자 DM 발송 + 쿨타임 적용) |
| `/내전예약 <시간>` | 일정 시간 후 자동 내전 개설 (예: `30분`, `1시간`) |
| `/아이디변경 <이름> <태그>` | 연동 라이엇 계정 변경 |
| `/내정보` | 내 계정·티어·라인 확인 |
| `/갱신` | 라이엇 API에서 내 정보 재조회 |
| `/팀짜기` | 현재 채널의 완성된 내전을 두 팀으로 자동 편성 |
| `/봇상태` | 봇 레이턴시·DB·Riot API 상태 확인 |

### 관리자 커맨드 (`/관리 ...`)

| 커맨드 | 설명 |
|---|---|
| `/관리 내전취소 <내전ID>` | 관리자가 특정 내전 강제 취소 |
| `/관리 유저추방 <@유저> [시간] [사유]` | 내전 참가 금지 (시간 없으면 영구) |
| `/관리 유저해제 <@유저>` | 내전 참가 금지 해제 |
| `/관리 알림역할 <@역할>` | 내전 알림 역할 설정 |
| `/관리 로그채널 <#채널>` | 에러 로그 채널 설정 |
| `/관리 쿨타임 <분>` | 재개설 쿨타임 설정 |
| `/관리 최대인원 <숫자>` | 내전 최대 인원 설정 |
| `/관리 설정보기` | 현재 서버 설정 확인 |

## 참가 플로우 (v2 - 포지션 선택 + 대기열)

```
✅ 참가하기 버튼 클릭
│
├─ 차단된 유저? ──────────────────────────────▶ 에러 메시지 (ephemeral)
├─ 이미 참가 중? ─────────────────────────────▶ 에러 메시지 (ephemeral)
│
├─ 내전 만석 (10/10)?
│   └─ 대기열에 추가 → "대기 X번입니다" 안내 (ephemeral)
│
└─ 자리 있음
    └─ 포지션 선택 드롭다운 표시 (ephemeral)
        │
        └─ 포지션 선택
            │
            ├─ 라이엇 ID 있음
            │   └─ 로비 추가 (선택 포지션 저장)
            │       └─ 임베드 갱신 → 10명 달성? → 완료 처리
            │
            └─ 라이엇 ID 없음
                └─ 모달 팝업 (게임이름 / 태그 입력, 포지션 유지)
                    └─ 제출 → Riot API 조회 → DB 저장 → 로비 추가
```

## 나가기 플로우 (v2 - 대기열 자동 승격)

```
❌ 나가기 버튼 클릭
│
├─ 개설자? ──────────▶ "취소 커맨드 사용 안내" (ephemeral)
│
└─ 로비에서 제거
    └─ 대기열 있음?
        └─ 첫 번째 대기자 자동 승격
            └─ DM 발송: "내전 자리가 생겼습니다!"
                └─ 임베드 갱신
```

```
티어 내림차순 정렬: [1위, 2위, 3위, 4위, 5위, 6위, 7위, 8위, 9위, 10위]

```

```
2차: 부라인으로 대체 (충돌 시)
3차: 남은 포지션으로 배정
```

## 봇 운영·안정성

### 백그라운드 태스크 (main.py)
| 태스크 | 주기 | 내용 |
|---|---|---|
| 만료 내전 정리 | 30분 | 24시간 이상 open 상태 내전 자동 취소 |
| 예약 내전 실행 | 1분 | fire_at 도달한 scheduled_lobbies 자동 개설 |

### 에러 로그
- 버튼/모달/커맨드 에러 발생 시 `log_channel_id` 채널에 자동 전송

### 쿨타임
- 내전 취소 후 `cooldown_minutes` (기본 5분) 동안 재개설 불가
- 서버별로 관리자가 조정 가능

- **Scopes**: `bot`, `applications.commands`

## 실행 방법

```powershell
# 1. 환경변수 설정
Copy-Item .env.example .env
# .env 열어서 DISCORD_TOKEN, RIOT_API_KEY 입력

# 2. 패키지 설치
pip install -r requirements.txt

# 3. 봇 실행
python main.py
```

> **주의**: Riot 개발자 API 키는 24시간마다 만료됩니다.  
> 봇을 장기 운영하려면 Riot Developer Portal에서 프로덕션 키를 신청해야 합니다.
